Пример на JavaScript: Создание базы данных IndexedDB.

ВНИМАНИЕ! Для проверки этого примера нужно использовать браузер Chrome и загружать данную страницу из репозитория Github. Если запускать ее локально, то пример работать не будет.

Предоставляется возможность создать БД различных версий:

  • Ver.1 - содержит одно хранилище объектов "Persons"
  • Ver.2 - содержит два хранилища объектов "Persons" и "Cars"
  • Ver.3 - содержит три хранилища объектов "Persons", "Cars" и "Orders"
Можно сразу создать базу данных версии Ver.2 или Ver.3, а можно сначала создать БД Ver.1, а затем обновить ее до версии Ver.2 или Ver.3. Перед каждым обновлением БД необходимо перезагружать страницу, т.е. после создания БД версии Ver.1, нужно перезагрузить страницу и обновить БД до Ver.2, затем еще раз перезагрузить страницу и обновить БД до Ver.3. По клавише F12 можно проверить создание БД с хранилищами объектов в браузере.

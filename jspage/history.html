Пример на JavaScript: Сохранение страниц в журнале посещений браузера.

После каждого изменения значений переменных сохраняйте страницу в журнале посещений браузера. Затем с помощью кнопок "->"(Вперед)" и ""<-"(Назад) браузера можно будет перемещаться по сохраненным страницам. При сохранении страниц, URL-адрес каждой страницы дополняется строкой #stateN, где N-значение переменной "x" в момент сохранения страницы. Все сохраненные страницы можно также просмотреть в журнале истории посещений браузера.

Пример использования Cookie:

Разверните на GlassFish-сервере сервлет из папки MyProjects_EE/CookieServlet или из папки Http_test/cookie_servlet/hello2. Возможно потребуется откорректировать URL используемого сервлета в файле CookiesPage.html. Удалите все Cookie в браузере. Затем введите свое имя и нажмите кнопку "Отправить". В результате будет отправлен ваш первый запрос на сервер и в ответ сервер вернет браузеру Cookie. Затем еще раз запустите эту страницу, и уже не вводя имени еще раз нажмите кнопку "Отправить". Теперь браузер отправит Cookie на сервер и сервер вернет страницу с приветствием.

Введите имя:

Нажмите кнопку чтобы создать Cookie для данной страницы с помощью JavaScript. Предварительно, в настройках браузера нужно разрешить сохранение сторонних Cookie. В браузере Chrome этот механизм создания Cookie не работал. В IE8 - все работало.
